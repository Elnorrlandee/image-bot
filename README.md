# Telegram AI Image Prompt Bot

بوت تيليجرام لتحويل وصف الصورة إلى برومبت محسن جاهز للاستخدام في مولدات الصور.

### مميزات
- دعم كامل للغة العربية والإنجليزية
- تحسين تلقائي للبرومبت (إضاءة، جودة، تفاصيل سينمائية)
- يدعم المحتوى +18 بدون قيود

### التشغيل محلياً
```bash
pip install -r requirements.txt
cp .env.example .env
python bot.py
